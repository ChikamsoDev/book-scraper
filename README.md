# Book Scraper

A Python web scraper that extracts book titles and prices from Books to Scrape and saves them into a CSV file.

## Features

- Scrapes 50 catalogue pages
- Extracts book titles
- Extracts prices
- Saves data as CSV
- Uses BeautifulSoup and Requests
- Includes polite scraping with delays

## Technologies

- Python
- Requests
- BeautifulSoup
- lxml
